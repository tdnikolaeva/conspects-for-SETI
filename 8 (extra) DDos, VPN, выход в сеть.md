* –û–±—Ä–∞–±–æ—Ç–∫–∞ –∑–∞–ø—Ä–æ—Å–∞:
  –í–µ–±-—Å–µ—Ä–≤–µ—Ä –∞–Ω–∞–ª–∏–∑–∏—Ä—É–µ—Ç –∑–∞–ø—Ä–æ—Å –∏ –æ–ø—Ä–µ–¥–µ–ª—è–µ—Ç, –∫–∞–∫–æ–π —Ä–µ—Å—É—Ä—Å —Ç—Ä–µ–±—É–µ—Ç—Å—è.

  –í–µ–±-—Å–µ—Ä–≤–µ—Ä —Ñ–æ—Ä–º–∏—Ä—É–µ—Ç HTTP-–æ—Ç–≤–µ—Ç.

–§–æ—Ä–º–∏—Ä–æ–≤–∞–Ω–∏–µ –æ—Ç–≤–µ—Ç–∞: –í–µ–±-—Å–µ—Ä–≤–µ—Ä —Ñ–æ—Ä–º–∏—Ä—É–µ—Ç HTTP-–æ—Ç–≤–µ—Ç, –∫–æ—Ç–æ—Ä—ã–π —Å–æ–¥–µ—Ä–∂–∏—Ç:

* –û—Ç–ø—Ä–∞–≤–∫–∞ –æ—Ç–≤–µ—Ç–∞: –í–µ–±-—Å–µ—Ä–≤–µ—Ä –æ—Ç–ø—Ä–∞–≤–ª—è–µ—Ç HTTP-–æ—Ç–≤–µ—Ç –≤ –≤–∏–¥–µ TCP-–ø–∞–∫–µ—Ç–æ–≤ —á–µ—Ä–µ–∑ —É—Å—Ç–∞–Ω–æ–≤–ª–µ–Ω–Ω–æ–µ TCP-—Å–æ–µ–¥–∏–Ω–µ–Ω–∏–µ.
* –î–ª—è HTTPS, –æ—Ç–≤–µ—Ç —Ç–∞–∫–∂–µ –∑–∞—à–∏—Ñ—Ä–æ–≤–∞–Ω —Å –ø–æ–º–æ—â—å—é TLS/SSL.

–ü–æ–ª—É—á–µ–Ω–∏–µ –∏ –æ–±—Ä–∞–±–æ—Ç–∫–∞ –æ—Ç–≤–µ—Ç–∞ –±—Ä–∞—É–∑–µ—Ä–æ–º
* –ü–æ–ª—É—á–µ–Ω–∏–µ –æ—Ç–≤–µ—Ç–∞: –ë—Ä–∞—É–∑–µ—Ä –ø–æ–ª—É—á–∞–µ—Ç TCP-–ø–∞–∫–µ—Ç—ã —Å HTTP-–æ—Ç–≤–µ—Ç–æ–º –æ—Ç —Å–µ—Ä–≤–µ—Ä–∞.
* –†–∞–∑–±–æ—Ä –æ—Ç–≤–µ—Ç–∞: –ë—Ä–∞—É–∑–µ—Ä –∞–Ω–∞–ª–∏–∑–∏—Ä—É–µ—Ç HTTP-–æ—Ç–≤–µ—Ç:
  –ü—Ä–æ–≤–µ—Ä—è–µ—Ç —Å—Ç–∞—Ç—É—Å-–∫–æ–¥.
  –†–∞–∑–±–∏—Ä–∞–µ—Ç –∑–∞–≥–æ–ª–æ–≤–∫–∏.
  –ü–æ–ª—É—á–∞–µ—Ç —Ç–µ–ª–æ –æ—Ç–≤–µ—Ç–∞.

–û—Ç–æ–±—Ä–∞–∂–µ–Ω–∏–µ –∫–æ–Ω—Ç–µ–Ω—Ç–∞: –ë—Ä–∞—É–∑–µ—Ä –æ–±—Ä–∞–±–∞—Ç—ã–≤–∞–µ—Ç HTML, CSS, JavaScript, –∏–∑–æ–±—Ä–∞–∂–µ–Ω–∏—è –∏ –¥—Ä—É–≥–∏–µ —Ä–µ—Å—É—Ä—Å—ã, –ø–æ–ª—É—á–µ–Ω–Ω—ã–µ –≤ —Ç–µ–ª–µ –æ—Ç–≤–µ—Ç–∞.

–ó–∞–∫—Ä—ã—Ç–∏–µ TCP-—Å–æ–µ–¥–∏–Ω–µ–Ω–∏—è (Four-way handshake)
* –ò–Ω–∏—Ü–∏–∞—Ü–∏—è –∑–∞–∫—Ä—ã—Ç–∏—è: –û–¥–Ω–∞ –∏–∑ —Å—Ç–æ—Ä–æ–Ω (–±—Ä–∞—É–∑–µ—Ä –∏–ª–∏ —Å–µ—Ä–≤–µ—Ä) –æ—Ç–ø—Ä–∞–≤–ª—è–µ—Ç TCP-–ø–∞–∫–µ—Ç —Å —É—Å—Ç–∞–Ω–æ–≤–ª–µ–Ω–Ω—ã–º —Ñ–ª–∞–≥–æ–º FIN, —á—Ç–æ–±—ã –∑–∞–∫—Ä—ã—Ç—å —Å–æ–µ–¥–∏–Ω–µ–Ω–∏–µ.
* –ü–æ–¥—Ç–≤–µ—Ä–∂–¥–µ–Ω–∏–µ –∑–∞–∫—Ä—ã—Ç–∏—è: –î—Ä—É–≥–∞—è —Å—Ç–æ—Ä–æ–Ω–∞ –æ—Ç–≤–µ—á–∞–µ—Ç TCP-–ø–∞–∫–µ—Ç–æ–º —Å —É—Å—Ç–∞–Ω–æ–≤–ª–µ–Ω–Ω—ã–º —Ñ–ª–∞–≥–æ–º ACK (–ø–æ–¥—Ç–≤–µ—Ä–∂–¥–∞–µ—Ç –ø–æ–ª—É—á–µ–Ω–∏–µ FIN).
* –û—Ç–ø—Ä–∞–≤–∫–∞ FIN: –î—Ä—É–≥–∞—è —Å—Ç–æ—Ä–æ–Ω–∞ –æ—Ç–ø—Ä–∞–≤–ª—è–µ—Ç TCP-–ø–∞–∫–µ—Ç —Å —Ñ–ª–∞–≥–æ–º FIN.
* –ü–æ–¥—Ç–≤–µ—Ä–∂–¥–µ–Ω–∏–µ –∑–∞–∫—Ä—ã—Ç–∏—è: –ü–µ—Ä–≤–∞—è —Å—Ç–æ—Ä–æ–Ω–∞ –æ—Ç–ø—Ä–∞–≤–ª—è–µ—Ç TCP-–ø–∞–∫–µ—Ç —Å —Ñ–ª–∞–≥–æ–º ACK, –ø–æ–¥—Ç–≤–µ—Ä–∂–¥–∞—è –ø–æ–ª—É—á–µ–Ω–∏–µ FIN.
* –°–æ–µ–¥–∏–Ω–µ–Ω–∏–µ –∑–∞–∫—Ä—ã—Ç–æ: TCP-—Å–æ–µ–¥–∏–Ω–µ–Ω–∏–µ –∑–∞–∫—Ä—ã—Ç–æ.

## DHCP

**DHCP** (Dynamic Host Configuration Protocol) ‚Äî —ç—Ç–æ —Å–µ—Ç–µ–≤–æ–π –ø—Ä–æ—Ç–æ–∫–æ–ª, –∫–æ—Ç–æ—Ä—ã–π –∞–≤—Ç–æ–º–∞—Ç–∏–∑–∏—Ä—É–µ—Ç –ø—Ä–æ—Ü–µ—Å—Å –Ω–∞–∑–Ω–∞—á–µ–Ω–∏—è IP-–∞–¥—Ä–µ—Å–æ–≤ –∏ –¥—Ä—É–≥–∏—Ö –ø–∞—Ä–∞–º–µ—Ç—Ä–æ–≤ –∫–æ–Ω—Ñ–∏–≥—É—Ä–∞—Ü–∏–∏ –¥–ª—è —É—Å—Ç—Ä–æ–π—Å—Ç–≤ –≤ —Å–µ—Ç–∏. –û–Ω –ø–æ–∑–≤–æ–ª—è–µ—Ç –∫–ª–∏–µ–Ω—Ç–∞–º –∞–≤—Ç–æ–º–∞—Ç–∏—á–µ—Å–∫–∏ –ø–æ–ª—É—á–∞—Ç—å –Ω–µ–æ–±—Ö–æ–¥–∏–º—ã–µ –Ω–∞—Å—Ç—Ä–æ–π–∫–∏ –ø—Ä–∏ –ø–æ–¥–∫–ª—é—á–µ–Ω–∏–∏ –∫ —Å–µ—Ç–∏, —á—Ç–æ –∑–Ω–∞—á–∏—Ç–µ–ª—å–Ω–æ —É–ø—Ä–æ—â–∞–µ—Ç —É–ø—Ä–∞–≤–ª–µ–Ω–∏–µ —Å–µ—Ç–µ–≤—ã–º–∏ —Ä–µ—Å—É—Ä—Å–∞–º–∏.

–û—Å–Ω–æ–≤–Ω—ã–µ —Ñ—É–Ω–∫—Ü–∏–∏ DHCP
* –ê–≤—Ç–æ–º–∞—Ç–∏—á–µ—Å–∫–æ–µ –Ω–∞–∑–Ω–∞—á–µ–Ω–∏–µ IP-–∞–¥—Ä–µ—Å–æ–≤: DHCP –ø–æ–∑–≤–æ–ª—è–µ—Ç —É—Å—Ç—Ä–æ–π—Å—Ç–≤–∞–º –ø–æ–ª—É—á–∞—Ç—å —É–Ω–∏–∫–∞–ª—å–Ω—ã–µ IP-–∞–¥—Ä–µ—Å–∞ –∏–∑ –∑–∞—Ä–∞–Ω–µ–µ –æ–ø—Ä–µ–¥–µ–ª–µ–Ω–Ω–æ–≥–æ –ø—É–ª–∞ –∞–¥—Ä–µ—Å–æ–≤.
* –ü–µ—Ä–µ–¥–∞—á–∞ –¥–æ–ø–æ–ª–Ω–∏—Ç–µ–ª—å–Ω—ã—Ö –ø–∞—Ä–∞–º–µ—Ç—Ä–æ–≤: –ü–æ–º–∏–º–æ IP-–∞–¥—Ä–µ—Å–∞, DHCP –º–æ–∂–µ—Ç –ø—Ä–µ–¥–æ—Å—Ç–∞–≤–ª—è—Ç—å –∏–Ω—Ñ–æ—Ä–º–∞—Ü–∏—é –æ –º–∞—Å–∫–µ –ø–æ–¥—Å–µ—Ç–∏, —à–ª—é–∑–µ –ø–æ —É–º–æ–ª—á–∞–Ω–∏—é, DNS-—Å–µ—Ä–≤–µ—Ä–∞—Ö –∏ –¥—Ä—É–≥–∏—Ö –Ω–∞—Å—Ç—Ä–æ–π–∫–∞—Ö —Å–µ—Ç–∏.
* –£–ø—Ä–∞–≤–ª–µ–Ω–∏–µ –∞—Ä–µ–Ω–¥–æ–π –∞–¥—Ä–µ—Å–æ–≤: DHCP –∏—Å–ø–æ–ª—å–∑—É–µ—Ç –º–µ—Ö–∞–Ω–∏–∑–º –∞—Ä–µ–Ω–¥—ã IP-–∞–¥—Ä–µ—Å–æ–≤, —á—Ç–æ –ø–æ–∑–≤–æ–ª—è–µ—Ç —ç—Ñ—Ñ–µ–∫—Ç–∏–≤–Ω–æ —É–ø—Ä–∞–≤–ª—è—Ç—å –∞–¥—Ä–µ—Å–Ω—ã–º –ø—Ä–æ—Å—Ç—Ä–∞–Ω—Å—Ç–≤–æ–º.

–ü—Ä–∏–Ω—Ü–∏–ø —Ä–∞–±–æ—Ç—ã DHCP (—Å—Ö–µ–º–∞ DORA)
–†–∞–±–æ—Ç–∞ –ø—Ä–æ—Ç–æ–∫–æ–ª–∞ DHCP –æ—Å–Ω–æ–≤–∞–Ω–∞ –Ω–∞ —á–µ—Ç—ã—Ä–µ—Ö —ç—Ç–∞–ø–∞—Ö, –∏–∑–≤–µ—Å—Ç–Ω–æ–π –∫–∞–∫ —Å—Ö–µ–º–∞ DORA:
1. Discovery (–û–±–Ω–∞—Ä—É–∂–µ–Ω–∏–µ):
    * –ö–ª–∏–µ–Ω—Ç –æ—Ç–ø—Ä–∞–≤–ª—è–µ—Ç —à–∏—Ä–æ–∫–æ–≤–µ—â–∞—Ç–µ–ª—å–Ω—ã–π –∑–∞–ø—Ä–æ—Å DHCPDISCOVER –¥–ª—è –ø–æ–∏—Å–∫–∞ –¥–æ—Å—Ç—É–ø–Ω—ã—Ö DHCP-—Å–µ—Ä–≤–µ—Ä–æ–≤ –≤ —Å–µ—Ç–∏.
2. Offer (–ü—Ä–µ–¥–ª–æ–∂–µ–Ω–∏–µ):
    * DHCP-—Å–µ—Ä–≤–µ—Ä—ã, –ø–æ–ª—É—á–∏–≤—à–∏–µ –∑–∞–ø—Ä–æ—Å, –æ—Ç–≤–µ—á–∞—é—Ç –∫–ª–∏–µ–Ω—Ç—É —Å–æ–æ–±—â–µ–Ω–∏–µ–º DHCPOFFER, –∫–æ—Ç–æ—Ä–æ–µ —Å–æ–¥–µ—Ä–∂–∏—Ç –ø—Ä–µ–¥–ª–æ–∂–µ–Ω–Ω—ã–π IP-–∞–¥—Ä–µ—Å –∏ –¥—Ä—É–≥–∏–µ –ø–∞—Ä–∞–º–µ—Ç—Ä—ã –∫–æ–Ω—Ñ–∏–≥—É—Ä–∞—Ü–∏–∏.
3. Request (–ó–∞–ø—Ä–æ—Å):
    * –ö–ª–∏–µ–Ω—Ç –≤—ã–±–∏—Ä–∞–µ—Ç –æ–¥–Ω–æ –∏–∑ –ø—Ä–µ–¥–ª–æ–∂–µ–Ω–∏–π –∏ –æ—Ç–ø—Ä–∞–≤–ª—è–µ—Ç —Å–µ—Ä–≤–µ—Ä—É —Å–æ–æ–±—â–µ–Ω–∏–µ DHCPREQUEST, –ø–æ–¥—Ç–≤–µ—Ä–∂–¥–∞—è —Å–≤–æ–µ –Ω–∞–º–µ—Ä–µ–Ω–∏–µ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å –ø—Ä–µ–¥–ª–æ–∂–µ–Ω–Ω—ã–π IP-–∞–¥—Ä–µ—Å.
4. Acknowledgement (–ü–æ–¥—Ç–≤–µ—Ä–∂–¥–µ–Ω–∏–µ):

## –ü—Ä–æ—Ç–æ–∫–æ–ª Ethernet

Ethernet ‚Äî —ç—Ç–æ —Å—Ç–∞–Ω–¥–∞—Ä—Ç –¥–ª—è –ª–æ–∫–∞–ª—å–Ω—ã—Ö —Å–µ—Ç–µ–π (LAN), –∫–æ—Ç–æ—Ä—ã–π –æ–ø—Ä–µ–¥–µ–ª—è–µ—Ç —Ñ–∏–∑–∏—á–µ—Å–∫–∏–µ –∏ –∫–∞–Ω–∞–ª—å–Ω—ã–µ —É—Ä–æ–≤–Ω–∏ –ø–µ—Ä–µ–¥–∞—á–∏ –¥–∞–Ω–Ω—ã—Ö. –û–Ω –∏—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è –¥–ª—è —Å–æ–µ–¥–∏–Ω–µ–Ω–∏—è —É—Å—Ç—Ä–æ–π—Å—Ç–≤ –≤ —Å–µ—Ç–∏ –∏ –æ–±–µ—Å–ø–µ—á–∏–≤–∞–µ—Ç –Ω–∞–¥–µ–∂–Ω—ã–π —Å–ø–æ—Å–æ–± –ø–µ—Ä–µ–¥–∞—á–∏ –¥–∞–Ω–Ω—ã—Ö –º–µ–∂–¥—É –Ω–∏–º–∏.

* –ü—Ä–æ—Ç–æ–∫–æ–ª—ã –¥–æ—Å—Ç—É–ø–∞ –∫ —Å—Ä–µ–¥–µ: Ethernet –∏—Å–ø–æ–ª—å–∑—É–µ—Ç –º–µ—Ç–æ–¥ CSMA/CD (Carrier Sense Multiple Access with Collision Detection), –∫–æ—Ç–æ—Ä—ã–π —É–ø—Ä–∞–≤–ª—è–µ—Ç –¥–æ—Å—Ç—É–ø–æ–º –∫ –æ–±—â–µ–π —Å—Ä–µ–¥–µ –ø–µ—Ä–µ–¥–∞—á–∏ –∏ –ø—Ä–µ–¥–æ—Ç–≤—Ä–∞—â–∞–µ—Ç –∫–æ–ª–ª–∏–∑–∏–∏.
* –°–∫–æ—Ä–æ—Å—Ç—å –ø–µ—Ä–µ–¥–∞—á–∏: Ethernet –ø–æ–¥–¥–µ—Ä–∂–∏–≤–∞–µ—Ç —Ä–∞–∑–ª–∏—á–Ω—ã–µ —Å–∫–æ—Ä–æ—Å—Ç–∏ –ø–µ—Ä–µ–¥–∞—á–∏ –¥–∞–Ω–Ω—ã—Ö, –≤–∫–ª—é—á–∞—è 10 Mbps (10BASE-T), 100 Mbps (Fast Ethernet), 1 Gbps (Gigabit Ethernet) –∏ –≤—ã—à–µ.
* –§–æ—Ä–º–∞—Ç –∫–∞–¥—Ä–æ–≤: –î–∞–Ω–Ω—ã–µ –ø–µ—Ä–µ–¥–∞—é—Ç—Å—è –≤ –≤–∏–¥–µ –∫–∞–¥—Ä–æ–≤, –∫–æ—Ç–æ—Ä—ã–µ —Å–æ–¥–µ—Ä–∂–∞—Ç –∑–∞–≥–æ–ª–æ–≤–æ–∫ —Å –∞–¥—Ä–µ—Å–∞–º–∏ –∏—Å—Ç–æ—á–Ω–∏–∫–∞ –∏ –Ω–∞–∑–Ω–∞—á–µ–Ω–∏—è, –∞ —Ç–∞–∫–∂–µ –∫–æ–Ω—Ç—Ä–æ–ª—å–Ω—É—é —Å—É–º–º—É –¥–ª—è –ø—Ä–æ–≤–µ—Ä–∫–∏ —Ü–µ–ª–æ—Å—Ç–Ω–æ—Å—Ç–∏ –¥–∞–Ω–Ω—ã—Ö.
